# ek-unique-search-cli-2

A CLI tool for searching text from files or standard input with unique filtering. Supports fixed string or regular expression patterns and case-insensitive search options.

## Features
- Search for a string or regex in files or stdin
- Only unique matching lines are printed
- Case-insensitive search option
- Flexible input: support for files and piped/stdin
